--- # Einführung in die PStA Korbinian Riedhammer --- # Praktische Studienarbeit: Formales - Separate Prüfungsteilleistung zum Modul _Programmieren 2_ - Keine Voraussetzung zur schriftlichen Prüfung - Nur im Sommer angeboten → bitte nicht schieben! - Sie sollten bereits angemeldet sein! - Einzelleistung, keine Teamarbeit --- # Praktische Studienarbeit: Durchführung - Die PStA ist als **Programmierwettbewerb** angelegt - Es gibt **36** kleine Aufgaben ("problems"), mit unterschiedlicher Schwierigkeit - Es müssen **32** Aufgaben gelöst werden, um zu bestehen. - [Domjudge](http://domjudge.inf.fh-rosenheim.de/) + System zum Hochladen und Bewerten der Lösungen (automatisiert) + Benutzername und Passwort per Email (Hochschuladdresse!) - Zeitlicher Rahmen: Contest ist + live ab **9.5. (heute) um 09:30 Uhr** + zu Ende am **3.7. 18:00 (s.t.)** --- # Domjudge Scoreboard - Jede abgegebene Lösung wird automatisch bewertet + öffentliche (bekannte) Testcases + geheime Testcases (deutlich ausführlicher) - Laufen alle Tests durch, so gilt das Problem als gelöst. + Löst man ein Problem **als erster**, so wird die Box **dunkelgrün** + Ansonsten wird die Box hellgrün - Man kann so viele Versuche hochladen wie man möchte - Je länger man braucht, um so mehr Strafpunkte gibt es (1 Punkt = 1 Minute, ab Contestbegin) - Wird eine falsche Lösung abgegeben, so gibt es Strafpunkte (1 Tag) --- # Was tun bei Rejects? - Lesen Sie die Angabe **sehr** genau! - Die Testcases sind alle mehrfach überprüft - Überprüfen Sie Ihre Ausgabe -- Zeichen für Zeichen! - Wird Ihre Lösung nicht akzeptiert, ist sie zu 99.9% falsch - Sind Sie der Meinung, dass Ihre Lösung aber richtig ist, so können Sie eine **Clarification** beantragen. + in 99.9% der Fällen ist die Antwort: "Lösung falsch" :-) --- # Motivation .skip[ ] .skip[ ] ## 10 Kinogutscheine a 10 € (nach finaler Rangliste) --- # Kochrezept Die Aufgaben sind alle gleich gestrickt. - `public static void main(String[] args)` - Lesen von `System.in` - Schreiben nach `System.out` ``` class MySolution { public static void main(String[] args) { BufferedReader br = new InputStreamReader(System.in); // ... System.out.println("solution"); } } ``` --- # Problem `p1` #### Beschreibung Input: "foo" Output: "Hello foo!" #### Beispieltest Eingabe: ``` 3 world Jan SantaClaus ``` Ausgabe: ``` Hello world!
