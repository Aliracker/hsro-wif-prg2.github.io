--- # Problem `p3` #### Beschreibung Schreiben Sie ein Programm, welches eine feste Anzahl an Namen einliest, und diese dann begrüßt. #### Beispieltest Eingabe: ``` 1 World ``` Ausgabe: ``` Hello, World! ``` --- # 1. Versuch ```java import java.io.BufferedReader; import java.io.IOException; import java.io.InputStreamReader; public class Greeter { public static void main(String[] args) throws IOException { BufferedReader br = new BufferedReader(new InputStreamReader(System.in)); int anz = Integer.parseInt(br.readLine()); for (int i = 0; i < anz; i++) { String name = br.readLine(); System.out.println("Hello, " + name); } } } ``` ---
